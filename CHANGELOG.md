# 更新日志 (Changelog)

所有显著的代码变更都将记录在此文件中。

## [2.4.0] - 2025-02-24

### ✨ 新增
- **多文件批量上传**：支持文件对话框多选、拖拽多选以及剪贴板批量图像粘贴上传。
- **队列处理机制**：引入前端串行队列逻辑，逐个上传文件，确保超大批次上传的稳定性。
- **实时进度反馈**：上传时实时显示当前队列进度（如 1/5）。
- **批量入库**：批量上传成功的图片会自动全数记录在本地画廊中。

### 🔧 优化
- 更新 UI 文案，明确提示支持批量操作。
